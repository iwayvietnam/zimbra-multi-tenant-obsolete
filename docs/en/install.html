Installation guide

Drupal CMS

First, you need install Drupal CMS (see link download and install like every CMS): with Databse,Hosting, user information,...
Now, you need to install ZaaSmhst module, see link download.
Log in with root account into Drupal CMS. From Management menu, choose Administration -> Modules -> Click Install new module
In this form, show file module address (or upload from local). and click Install

Zimbra server
Simply, Copy phpchecker file to zimbra server anywhere you can access and run it in crontab
