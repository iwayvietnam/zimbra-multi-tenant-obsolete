Quản trị Zimbra


Đây là trang thông tin cho phần mềm ZaaS.
ZaaS là phần mở rộng tính năng cho phiên bản Zimbra Mã nguồn mở. Zimbra as a Service sử dụng giao diện là web Drupal để tương tác giữa người dùng là các quản trị domain với hệ thống mail server Zimbra. Các tính năng của phần mềm:
System Admin - Quản trị hệ thống: Quản lý định mức, Quản lý tenant, Lên kế hoạch thực thi (từ trong CSDL).
Domain Admin: Quản trị hòm thư (mailbox), Quản lý alias, Quản lý group (Distributions list)
Tenant: Quản trị domain
