Hướng dẫn cài đặt

Drupal CMS

Trước tiên, bạn cần cài đặt Hệ thống quản trị nội dung Drupal ( Drupal CMS). Xem link cài đặt và cách cài đặt cũng giống như các CMS khác, với các thông số như tên CSDL, mật khẩu, địa chỉ host, thông tin cá nhân, người sử dụng,...
Bây giờ, bạn cần cài đặt module ZaaS, xem link download
Đăng nhập bằng tài khoản ROOT vào Drupal CMS. Từ menu Management, chọn Administration -> Modules -> Nhấn vào Install new module
Ở mục này, chỉ ra đường dẫn của module hoặc tải lên từ máy tính của bạn, rồi Nhấn vào Install

Máy chủ Zimbra
Rất đơn giản, sao chép file phpchecker lên máy chủ zimbra ở bất cứ chỗ nào mà bạn có thể truy cập và chạy nó bằng crontab
